- فهرس السور مع معلومات السورة
- عرض الآيات مع أرقامها
- تشغيل تلاوة للسورة كاملة
- تشغيل تلاوة لآية واحدة بالضغط عليها

- وضع داكن
- وضع صفحة كلاسيكي
